# Kerry Fleet Dashboard

Vite + React dashboard ready for Vercel.

## Local run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Push this folder to GitHub.
2. Import the repository in Vercel.
3. Framework preset: Vite.
4. Build command: `npm run build`.
5. Output directory: `dist`.

The app calls:

```txt
https://green-route-optimization.onrender.com/ask-zone/{dcspName}
```

SPDP and SPYR are mapped to local mock data first.
